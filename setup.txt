## 1️⃣ Install `uv`

`uv` is a fast Python package and environment manager.

Run this command in the terminal:

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

---

## 2️⃣ Restart the Terminal

After installing `uv`, restart your terminal
**OR** run:

```bash
source ~/.bashrc
```

---

## 3️⃣ Verify Installation

Check that `uv` is installed correctly:

```bash
uv --version
```

---

## 4️⃣ Create a Virtual Environment (Python 3.10)

Create a virtual environment using Python 3.10:

```bash
uv venv --python 3.10
```

This will create a folder named `.venv`.

---

## 5️⃣ Activate the Virtual Environment

Activate the environment before installing dependencies.

```bash
source .venv/bin/activate
```

---

## 6️⃣ Confirm Python Version

Make sure Python 3.10 is being used:

```bash
python --version
```

Expected output:

```
Python 3.10.x
```

⚠️ **Important:**
Do **NOT** close the terminal or deactivate the environment while working on the project.

---

## 7️⃣ Install Project Dependencies

Install all required packages from `requirements.txt`:

```bash
uv pip install -r requirements.txt
```

---

## 8️⃣ Collect Training Data

Run the training script to collect gesture data:

```bash
python train.py
```

---

## 9️⃣ Run Gesture Detection

After training is complete, run the detection script:

```bash
python detect_gesture.py
```

---

## ✅ You’re Done!

Your environment is now set up and the gesture detection system should be working 🎉
